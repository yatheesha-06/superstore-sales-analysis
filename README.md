# Superstore Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing Superstore sales data using Microsoft Excel and Python. The objective of the project is to identify sales trends, top-performing products, regional performance, and category-wise sales insights through interactive dashboards and visualizations.

The project includes:
- Excel Dashboard with Pivot Tables, Charts, and Slicers
- Python Data Analysis using Pandas and Matplotlib
- Sales Trend Analysis
- Top Product Analysis
- Regional Sales Analysis

---

- Slicers & Dashboard Design
- Python

## 📊 Features
- Interactive Excel Dashboard
- Sales by Region Analysis
- Sales by Category Analysis
- Top 10 Products Analysis
- Monthly Sales Trend Visualization
- Pie Charts, Bar Charts, and Line Charts
- Data Cleaning and Exploration using Python
